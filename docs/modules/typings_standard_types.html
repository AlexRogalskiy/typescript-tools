  • isFalsy(value: unknown): value is Falsy
    • Defined in typings/standard-types.ts:197

    Tests for Falsy by simply applying negation ! to the tested val.

    The value is mostly in added type-information and explicitly, but in case of this simple type much the same can often be archived by just using negation !:

    example

    const consumer = (value: boolean | Falsy) => { if (!value) { return ; } type newType = typeof value; // === true // do stuff };

Const isPrimitive

  • isPrimitive(value: any): value is Optional<string | number | bigint | boolean | symbol>
    • Defined in typings/standard-types.ts:29

    Tests for one of the Primitive types using the JavaScript typeof operator

    Clarification: TypeScript overloads this operator to produce TypeScript types if used in context of types.

    example

    const consumer = (value: Primitive | Primitive[]) => { if (isPrimitive(value)) { return console.log('Primitive value: ', value); } // type of value now inferred as Primitive[] value.map((primitive) => consumer(primitive)); };

    Parameters

    • value: any

      The value to be tested

    Returns value is Optional<string | number | bigint | boolean | symbol>

    If val is primitive. If used in the flow of the program typescript will infer type-information from this.
